# Connect-Cloudant-Store sample with App ID

## Overview

Sample application which use Connect-Cloudant-Store with IBM AppID.


## Result

AppID のログイン自体は成功しているっぽいが、そこから画面が遷移せず、ずっとログイン画面のまま。ログなし。

セッション設定で Cloudant を使わなければログイン可能。
